### Information 

Tanken är att med 119 skapa grupper.

Dessa hanteras sedan med 121, oberoende av medlemssystemet.

När turneringen är klar ska resultaten tillbaks till medlemssystemet.

För att kunna mata in resultaten i samma ordning, krävs att samma Berger-tabell används.

Det innebär också att slumpning av deltagarna inte kan tillåtas.

Vilket naturligtvis är obra.

Slumpning av Berger är inte lika viktig som slumpning av Schweizer.

Jag ser ingen väg runt detta, förutom att låta bli att publicera turneringen i medlemssystemet. Vilket också är obra.

Orsaken till att testturneringar inte läggs upp på testsystemet är att bookmarklets inte fungerar där. Urlen skiljer sig.
